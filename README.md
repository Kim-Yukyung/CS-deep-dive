### 진행 방식

1. 폴더 & 파일 구조
각 챕터 폴더(챕터1, 챕터2…) 아래에 {이름}/, {이름}.md 형식으로 자유롭게 정리

2. 요약 발표
매주 한 명이 돌아가며 해당 챕터를 요약해 발표

3. 한 줄 질문
매주 최소 1개씩 질문 또는 토론거리 준비 → 발표 시간에 함께 공유하고 이야기 나누기
